# Case Management & Service Automation System

A customized Dynamics 365 / Power Platform solution built to automate internal service desk workflows, track core client SLAs, and manage customer support escalations efficiently.

##  Key Features

*   **Case Resolution Lifecycle (`ResolveCase`):** Client-side scripts and form logic optimized for streamlined support ticket handling and closing sequences.
*   **SLA Milestone Monitoring (`ShowMilestone`):** Automated visual indicators and tracking configurations to ensure key compliance windows are consistently met.
*   **Sponsorship Verification (`CheckSponsor`):** Specialized verification logic to validate account priority levels, client backing, and high-tier escalation requirements.
*   **Metric Aggregation:** Backend rollup rules configured to calculate and monitor case handling parameters dynamically.

##  Repository Structure

*   `/src`: The extracted, human-readable source components of the solution customization files, plugins, and client-side JavaScript assets.
*   `/release`: The deployment-ready managed/unmanaged `.zip` solution container files.
